You've activated the Full vibrational properties in-app guide. In this guide, you will learn how to use the app to compute the phonons, inelastic neutron scattering, Raman/IR spectra and dielectric properties (Dielectric and Raman tensor, Born effective charges) of bulk Silicon (Si).
Pre-requisite: This guide assumes that you already followed the general → basic guide, or that you are already familiar with the app.
Click on Step 1: Select structure and follow the instructions to proceed.
In the structure selection step, we select the Si structure from the examples.

Tasks

  • Click on the "From examples" tab
  • Select the Silicon structure in the dropdown menu
  • Click the Confirm button to proceed
Warning: If the confirmed structure is not yet stored in the AiiDA database, it will be stored automatically when you proceed to the next step.
Warning: Changes after confirmation will reset the following steps.
In the configuration step, we define the parameters needed to perform the calculations.

Tasks

  1. Select Full geometry, i.e. we relaxed the structure optimizing both cell parameters and atomic positions
  2. Open Step 2.1 for further instructions
Here we select the properties to calculate. Each property is associated with a workflow that will submit one or more calculations.

Tasks

  1. Tick the Vibrational Spectroscopy (VIBRO) checkbox
  2. Open Step 2.2 for further instructions
Here we can customize the calculation parameters. The settings are divided into several tabs:
  • Basic settings: top-level calculation settings
  • Advanced settings: fine-tune the calculations
  • Plugin settings: parameters for plugin calculations including phonon settings (Vibrational settings)
Note that the app pre-configures defaults for most parameters.
Here we define the basic settings of the simulations.

Tasks

  1. Select Insulator Electronic type
  2. Select Fast protocol
  3. See further instructions in the Vibrational settings panel
Note: For the purpose of completing this guide in short time, we will use the Fast protocol for default parameters. However, you if dispose of enough computational resources, you can also select Moderate or Precise.
In the Vibrational settings panel, we can customize the parameters for the Raman+IR calculation.

Tasks

  • Select IR/Raman, Phonon, Dielectric, INS properties in the dropdown menu.
  • Set the Supercell size to be [2,2,2], if not already set.
  • Estimate the number of supercells to see the number of supercells needed to computed finite displacement forces. The output number should be equal to 1, thanks to symmetry between different displacements.
  • Click the Confirm button to proceed
Note: a [1,1,1] supercell (i.e., unit cell) is not accurate to correctly reproduce phonons dispersion, but is fine for phonons at Γ. Usually, at least a [2,2,2] supercell is needed.
Warning: Changes after confirmation will reset the following steps.

In the submission step, we define the computational resources to be used in the calculations. The global resources are used to define resources across all workflow calculations. Optionally, you can override the resource settings for specific calculations.

Warning: If running locally (for example, on the AiiDAlab demo server), we recommend keeping nodes and CPUs at the default minimum of 1 each.

Once the resources are defined, we can optionally customize the workflow label (pre-filled according to the settings of steps 1 & 2), as well as provide a detailed description of the workflow. Once we are ready, we can submit the workflow.

Tasks

  1. Select 1 node and 1 CPU for each of the codes below
  2. (Optional) customize the workflow label
  3. (Optional) add a workflow description
  4. Click the Submit button to proceed
Warning: The workflow may take a moment to submit.
In the results panel, each tab will open results pertaining to a specific calculation submitted by the workflow.
See further instructions below in the Vibroscopy results tab. Once the calculation is complete, click on the Load results button to load the available results.

Tasks

  1. Inspect the three quantities represented in the tables: Dielectric, Raman tensors and Born effective charges.
  2. Use the Select atom site dropwdown to inspect the second Si atom.
Here we obtain the dynamical structure factor (from inelastic neutron scattering INS) spectrum S(Q, ω).

Tasks

  1. Click on the Initialise INS data to start computing the INS signal.
  2. Change units to be THz.
  3. Decrease the q step to be 0.01 Å-1.
  4. Increase the number of energy bins #E bins to be 1000.
  5. Click on the Replot to perform again the calculation.
  6. Select the Plot mode to be DOS map, replot and verify that you can find the same degenerate mode seen in the phonons spectrum.
  7. Play with the other parameters to change the plot as you prefer.
